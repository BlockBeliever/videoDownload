# videoDownload
 一个一键下载vip视频的软件
[下载地址](https://qqhxj.oss-cn-beijing.aliyuncs.com/vipVideoDownloader.exe)
## 功能说明

1. 下载`vip`视频
2. 下载`m3u8`格式视频
3. 多线程下载视频
4. 支持在线搜索
5. 支持电视剧多集下载

# 如何使用
1. 双击exe可执行文件
2. 在命令行界面输入命令

## 命令介绍

>用法: 命令 + 参数
>
> d [ url | [ vod_id [opt] ]               下载视频                          
> s [word]                                 搜索视频                          
> detail [vod_id]                          详细视频
